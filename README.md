 # 🔧 Ferretería JM

Sistema web para la administración de productos y categorías de una ferretería.

Desarrollado como proyecto final utilizando una arquitectura Monorepo con Astro, React, PHP, PDO, MySQL y Docker.

---

# 🚀 Tecnologías

## Frontend

- Astro
- React
- JavaScript
- CSS

## Backend

- PHP 8
- PDO
- API REST

## Base de datos

- MySQL

## Infraestructura

- Docker
- Docker Compose

---

# ⚙ Instalación

## Clonar el proyecto

```bash
git clone https://github.com/TU-USUARIO/FERRETERIA.git
```

Ingresar al proyecto

```bash
cd ferreteria-jm
```

Levantar los contenedores

```bash
docker compose up -d
```

# ✅ Funcionalidades

- Inicio dinámico.
- Login de administrador.
- Dashboard administrativo.
- CRUD de Categorías.
- CRUD de Productos.
- Carga de imágenes.
- Vista previa de imágenes.
- Carrusel de Ofertas Especiales.
- Navbar responsive.
- Footer responsive.
- API REST en PHP.
- Base de datos MySQL.
